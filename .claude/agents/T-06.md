## ATTENZIONE CRITICA — Timezone cross-platform
- Windows: TimeZoneInfo.FindSystemTimeZoneById("Eastern Standard Time")
- Linux:   TimeZoneInfo.FindSystemTimeZoneById("America/New_York")

Usa questa logica:
```csharp
private static TimeZoneInfo GetEasternTimeZone() =>
    OperatingSystem.IsWindows()
        ? TimeZoneInfo.FindSystemTimeZoneById("Eastern Standard Time")
        : TimeZoneInfo.FindSystemTimeZoneById("America/New_York");
```

Aggiorna skill-dotnet.md con questo pattern se non già presente.
